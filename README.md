# 💳 Sistema de Contas Bancárias (SQL)

Projeto simples de manipulação de dados utilizando SQL.

## 📌 Funcionalidades
- Criação de tabela
- Inserção de dados
- Atualização de saldo
- Transferência entre contas (transações)
- Uso de COMMIT e ROLLBACK
- Consultas com filtros
- Ordenação de dados

## 🛠 Tecnologias
- MySQL
- SQL

## ▶️ Como executar
1. Abra o MySQL Workbench
2. Execute o arquivo `script.sql`
